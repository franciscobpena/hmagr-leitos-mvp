Política de Privacidade

Última atualização: abril de 2026

Dados exibidos

Este dashboard exibe dados operacionais anonimizados do Hospital Estadual Sílvio Ávidos (HMSA), Colatina/ES. Não são exibidos nome de pacientes, médicos, CPF ou qualquer dado pessoal identificável. Os dados provêm da aba AGREGADO_PUB do sistema de gestão de leitos, gerada automaticamente pelo Apps Script com remoção de dados sensíveis.

Anotações

As anotações de barreira, ação e responsável registradas neste dashboard são armazenadas em banco de dados seguro (Supabase, região São Paulo) e localmente no dispositivo (localStorage). Não são coletados dados pessoais de pacientes nestas anotações.

Cookies e rastreamento

Este sistema não utiliza cookies de rastreamento, anúncios ou análise de comportamento de terceiros.
